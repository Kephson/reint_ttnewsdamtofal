What does it do?
================

This extension converts the DAM <media> tags in the bodytext element of tt_news entries to the corresponding FAL <link> element to update the entries in TYPO3 6.2.
The extension provides a backend module in TYPO3 to start to conversion of the entries.

If there are any questions or changes please contact me (ephraim.haerer@renolit.com).
